## ⚙️ التثبيت والتشغيل

### المتطلبات
- Node.js 18+
- npm
- (اختياري) ADB لربط الهاتف

### خطوات التثبيت

```bash
# 1. استنساخ المشروع
git clone https://github.com/your-username/cashy-link.git
cd cashy-link

# 2. تثبيت المكتبات
npm install

# 3. إنشاء ملف البيئة
cp .env.example .env
# عدّل .env وضع JWT_SECRET قوياً

# 4. تشغيل الخادم
npm start

# أو للتطوير مع إعادة تشغيل تلقائية
npm run dev
```

الموقع سيعمل على: **http://localhost:3000**

**بيانات الدخول الافتراضية:** `admin` / `admin123`

> ⚠️ غيّر كلمة مرور المدير فور التشغيل الأول!

## 📱 ربط الهاتف عبر ADB

### المتطلبات
1. تفعيل **خيارات المطور** في الهاتف (اضغط على رقم الإصدار 7 مرات)
2. تفعيل **تصحيح أخطاء USB** في خيارات المطور
3. تثبيت **Android Platform Tools** على الكمبيوتر

### تثبيت ADB

**Windows:**
```bash
winget install Google.PlatformTools
# أو تحميل من: https://developer.android.com/tools/releases/platform-tools
```

**Linux:**
```bash
sudo apt install android-tools-adb
```

**macOS:**
```bash
brew install android-platform-tools
```

### الربط

```bash
# تحقق من اتصال الهاتف
adb devices

# يجب أن يظهر الجهاز كـ "device"
# قبول رسالة السماح على الهاتف
```

### إضافة ADB في ملف .env
```env
ADB_PATH=adb   # أو المسار الكامل مثل: C:\platform-tools\adb.exe
```

## 🚀 المميزات الرئيسية

### 💼 إدارة المحافظ
- إضافة وتعديل محافظ متعددة لكل مزود
- تتبع الرصيد في الوقت الفعلي
- الحد اليومي والشهري مع تنبيهات تلقائية
- ربط المحفظة بهاتف Android

### 💸 العمليات المالية
- **إرسال** — تحويل أموال لعميل
- **استلام** — استقبال تحويل
- **سحب** — سحب نقدي
- **إيداع** — إيداع نقدي
- حساب الرسوم تلقائياً حسب المزود
- إلغاء العملية وإعادة الرصيد

### 📊 الإحصاءات
- لوحة تحكم مباشرة بالأرقام والرسوم البيانية
- تقرير يومي مفصل لكل محفظة
- تحليل حسب المزود
- توزيع ساعي للعمليات
- أفضل العملاء

### 📱 ربط الهاتف (ADB)
- عرض الأجهزة المتصلة
- جلب رسائل SMS وتحليلها تلقائياً
- إرسال أكواد USSD للاستعلام عن الرصيد
- ربط كل جهاز بمحفظة محددة

### 👥 إدارة العملاء
- سجل كامل بالعملاء وتاريخ معاملاتهم
- بحث سريع بالاسم أو الهاتف

### 📤 التقارير والتصدير
- تصدير Excel مفصل وملوّن
- تقرير يومي بكل المحافظ
- طباعة مباشرة

## 🏗️ التقنيات المستخدمة

**Backend:**
- Node.js + Express
- SQLite (better-sqlite3) — لا يحتاج إعداد قاعدة بيانات خارجية
- JWT للمصادقة
- WebSocket للتحديث الفوري
- ExcelJS لتصدير التقارير

**Frontend:**
- HTML/CSS/JavaScript خالص (لا framework)
- Chart.js للرسوم البيانية
- Font Awesome للأيقونات
- خط Cairo (عربي)
